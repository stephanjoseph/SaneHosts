Privacy Policy

Last updated: January 2026

Summary

SaneHosts does not collect, transmit, or store any personal data. Your hosts file configurations and profiles remain entirely on your Mac.

Data Collection

SaneHosts does not collect:

  • Personal information
  • Usage analytics
  • Crash reports
  • Telemetry of any kind

Local Storage

SaneHosts stores the following data locally on your Mac:

  • Profiles: ~/Library/Application Support/SaneHosts/Profiles/
  • Backups: ~/Library/Application Support/SaneHosts/Backups/
  • Preferences: Standard macOS UserDefaults

This data never leaves your computer.

Network Access

SaneHosts accesses the network only when:

  • You explicitly import a blocklist from a remote URL
  • Checking for app updates via Sparkle (can be disabled in Settings)

No data about your profiles or hosts configurations is ever transmitted.

System Access

SaneHosts requires administrator privileges to modify /etc/hosts. This is handled through macOS's standard authentication dialog. We do not store or have access to your admin password.

Third-Party Services

SaneHosts uses:

  • Sparkle: For checking app updates. Only checks for updates from sanehosts.com. No personal data is transmitted.

Open Source

SaneHosts is open source. You can review the code at github.com/sane-apps/SaneHosts.

Contact

Questions about privacy? Open an issue on GitHub.
