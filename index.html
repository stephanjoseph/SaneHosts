What is a hosts file?

Every time your Mac tries to connect to a website, it first checks a local file called /etc/hosts. This file can redirect domains to different addresses, or block them entirely.

By adding ad servers and trackers to this file, you can block them at the system level, before they even load. No browser extension needed, works in every app.

SaneHosts makes managing this file simple and safe, with profiles you can switch between instantly.

# Block ads and trackers
0.0.0.0 ads.example.com
0.0.0.0 tracker.analytics.com
0.0.0.0 pixel.facebook.com

# Local development
127.0.0.1 myapp.local
127.0.0.1 api.myapp.local
